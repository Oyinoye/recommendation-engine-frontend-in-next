Frontend Requirements
Use Next.js (App Router) and Tailwind CSS.
Real-time Dashboard: A view showing a live stream of ingested events. (Hint: Consider WebSockets, SSE, or Firestore real-time listeners).
Persona Viewer: A page where a user can search for a user_id and see their AI-generated persona/context.
Standardized UI: Use a consistent design system (e.g., Shadcn/ui) with high accesssibility scores (90+ Lighthouse).
Optimistic Updates & Error Handling: The UI should reflect state changes immediately and gracefully handle API failures or timeouts from the LLM.
